= 1.1.2 Release - February 02, 2026



== Release Notes & Highlights



Detailed information about the most important changes in 1.1.2.

This section focuses on changes that directly impact users and require attention during upgrades.



[[note-1_1_2-SEC-001-patched-xss-vulnerabil]]
[.release-note]
--
Fixed a cross-site scripting (XSS) vulnerability in user input handling.
This security patch prevents malicious scripts from being executed through user-generated content.

--


[[note-1_1_2-BUG-004-fixed-database-connect]]
[.release-note]
--
Fixed database connection timeout issues that could occur during high-traffic periods.
The system now handles concurrent connections more efficiently.

--


[[note-1_1_2-BUG-005-resolved-keyboard-shor]]
[.release-note]
--
Resolved keyboard shortcut conflicts with Firefox browser.
All application shortcuts now work correctly across different browsers.

--


[[note-1_1_2-IMPROVE-003-optimized-search-index]]
[.release-note]
--
Optimized search indexing process, reducing indexing time by up to 60% for large datasets.
Search results now appear faster and system responsiveness is improved.

--


[[note-1_1_2-IMPROVE-004-added-progress-indicat]]
[.release-note]
--
Added progress indicators for long-running operations including data exports, bulk updates, and large file uploads.
Users now receive real-time feedback on operation status.

--


[[note-1_1_2-DOC-001-updated-api-documentat]]
[.release-note]
--
Updated API documentation with comprehensive OAuth2 migration guide and examples.
Includes step-by-step instructions for transitioning from basic authentication.

--
