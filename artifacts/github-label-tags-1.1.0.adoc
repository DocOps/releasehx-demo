= Release History -- 1.1.0 - 2026-02-02


== Release Notes


Descriptions of any specially notable changes or additions since the previous release.


[[note-1_1_0-42]]
[.release-note]
--
Introduced a dark mode toggle in user preferences, allowing users to switch between light and dark themes.
The setting is automatically saved and persists across sessions.

--


[[note-1_1_0-43]]
[.release-note]
--
Fixed a memory leak in the background synchronization process that was causing gradual performance degradation.
Users should notice improved stability during long sessions.

--


[[note-1_1_0-44]]
[.release-note]
--
Implemented OAuth2 authentication flow to replace basic authentication.
This provides enhanced security and enables seamless integration with third-party services.

*Breaking Change:* Basic authentication will be deprecated in v1.2.0.
Please migrate to OAuth2 before then.

--


[[note-1_1_0-45]]
[.release-note]
--
Added keyboard shortcuts for common actions:

* `Ctrl+N` / `Cmd+N`: Create new item
* `Ctrl+S` / `Cmd+S`: Save current work
* `Ctrl+F` / `Cmd+F`: Open search
* `Esc`: Close modals/dialogs

--


[[note-1_1_0-46]]
[.release-note]
--
The following legacy API endpoints are now deprecated and will be removed in v1.3.0:

* `/api/v1/users` → Use `/api/v2/users` instead
* `/api/v1/projects` → Use `/api/v2/projects` instead
* `/api/v1/tasks` → Use `/api/v2/tasks` instead

The new endpoints provide better performance and additional features.

--


[[note-1_1_0-47]]
[.release-note]
--
NO RELEASE NOTE PROVIDED

--
