Release 1.1.0 - 2026-02-02

Notable Changes

Detailed information about significant changes that may affect users.

Highlighted Features

Collaboration

Introduced real-time collaboration features allowing multiple users to edit documents simultaneously. Changes are synchronized instantly across all connected clients with conflict resolution.

Search & Discovery

Added advanced search capabilities with filtering options including: - Date ranges - Content types - User assignments - Status filters - Custom tags Search results are now more precise and easier to navigate.

AI Features

Introduced experimental AI-powered content suggestions to help users create better content. This feature: - Suggests improvements to writing - Recommends relevant tags - Provides content structure guidance **Note:** This is an experimental feature and may change in future releases.

Breaking Changes

Configuration

Removed the following deprecated configuration options: - `legacy_mode` (use `compatibility.legacy_support` instead) - `old_api_format` (use `api.response_format` instead) - `deprecated_auth` (OAuth2 is now required) Please update your configuration files accordingly.

Removed Features

Configuration

Removed the following deprecated configuration options: - `legacy_mode` (use `compatibility.legacy_support` instead) - `old_api_format` (use `api.response_format` instead) - `deprecated_auth` (OAuth2 is now required) Please update your configuration files accordingly.

Experimental Features

AI Features

Introduced experimental AI-powered content suggestions to help users create better content. This feature: - Suggests improvements to writing - Recommends relevant tags - Provides content structure guidance **Note:** This is an experimental feature and may change in future releases.
