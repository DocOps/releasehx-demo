Added inline editing to dashboard widgets

Application Programming Interface

Added batch export API endpoint
A new batch export endpoint is now available at `/api/export/batch`. Supported formats: - CSV (default) - JSON with optional compression Example usage: ```bash curl -X POST /api/export/batch \ -H "Content-Type: application/json" \ -d '{"format": "csv", "ids": [1,2,3]}' ```
